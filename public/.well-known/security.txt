# Bug Bounty Program
# We operate a private bug bounty program for security researchers.
# Please contact us for program details and scope.

# Disclosure Policy
# We follow coordinated disclosure practices.
# Please allow 90 days for remediation before public disclosure.

# Legal
# By testing and reporting vulnerabilities, you agree to:
# 1. Not access or modify user data without explicit permission
# 2. Not perform denial of service attacks
# 3. Not use automated scanners that may impact service availability
# 4. Not disclose vulnerabilities to third parties before we've had time to fix them
# 5. Comply with all applicable laws and regulations

# Safe Harbor
# We consider security research conducted in accordance with this policy
# to be authorized conduct. We will not initiate legal action against
# researchers who discover, test, and report vulnerabilities in accordance
# with this policy.

# Thank you for helping keep PetPark and our users safe!
