# PetPark Copy System hr-HR v2.0

## 1. Voice

PetPark copy should sound:

```txt
warm
clear
local
trustworthy
helpful
human
not corporate
not salesy
```

Primary tone: friendly Croatian product language.

Avoid overly formal, generic marketplace language. Pet owners should feel guided, not sold to.

## 2. Core phrases

Approved brand/product phrases:

```txt
Što treba tvom ljubimcu danas?

Brzo, sigurno i s ljubavlju.

Sve na jednom mjestu.

```

## 5. Service names

Use these labels consistently:

| Service code | Croatian label       | Notes                                                           |
| ------------ | -------------------- | --------------------------------------------------------------- |
| `boarding`   | Čuvanje              | Includes temporary care/sitting.                                |
| `walking`    | Šetnja               | Dog walking / active walks.                                     |
| `grooming`   | Grooming             | Keep Croatian market term; optionally explain as njega/šišanje. |
| `training`   | Trening              | Training/behavior guidance.                                     |
| `lost_found` | Izgubljeni/pronađeni | Use full label outside orbit when space allows.                 |
| `adoption`   | Udomljavanje         | Adoption/rescue.                                                |

Orbit labels may be shorter:

```txt
Čuvanje
Šetnja
Grooming
Trening
Izgubljeni
Udomljavanje
```

## 6. Trust copy

Preferred platform-level trust language:

```txt
Sve na jednom mjestu
Sigurniji dogovor
Zajednica koja pomaže
Provjeravamo korisnike i recenzije za mir bez brige.
Pouzdane opcije i jasna komunikacija.
```

Avoid repetitive provider-directory trust language such as:

```txt
Provjereni pružatelji
Provjereni čuvari
Samo provjereni
```

It can be used carefully as a platform note, not a homepage section.
